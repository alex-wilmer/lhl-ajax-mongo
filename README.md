## Ajax, Express, Mongo

### Quickstart

Install:

```
npm i
```

Make sure the MongoDB Daemon is running by entering your shell:

```
mongo
```

If that command does not work try starting your daemon with:

```
mongod
```

Start the server:

```
npm run server
```

Start the client:

```
npm run client
```

Visit http://localhost:1234 in your browser!
